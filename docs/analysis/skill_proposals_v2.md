# GBase Skill 方案 v2.0 (Market-First)

> **原则**: 基于市场验证的需求，而非假设的痛点
> **日期**: 2026-01-10

---

## 设计原则

### 必须满足
1. **有验证的市场需求** - 客户已经在付费购买类似产品
2. **可量化的 ROI** - 买家能说清楚为什么付钱
3. **补充而非替代** - 增强 Salesforce 生态，不与之竞争
4. **嵌入工作流** - 不改变用户习惯

### 避免
1. 做没有付费验证的"痛点"产品
2. 与 Salesforce 原生功能竞争
3. 与成熟玩家 (DocuSign, Slack) 正面竞争
4. 忽视行业垂直机会

## Skill 1: Data Cloud Activator (数据云激活器)

### 市场验证
- Data Cloud ARR: **$900M** (Salesforce 最快增长产品)
- CDP 市场: $2.65B → $12.96B (CAGR 21.7%)
- 70% 企业 CRM 策略将依赖类似工具 (2026)

### 买家
| 角色 | 预算 | 购买动机 |
|------|------|----------|
| CDO (Chief Data Officer) | 数据预算 | 统一客户视图 |
| Marketing VP | MarTech 预算 | 受众激活、个性化 |
| IT Director | 集成预算 | 减少数据孤岛 |

### 产品定位
**"让 Data Cloud 数据真正产生价值"**

不是替代 Data Cloud，而是：
- 帮助企业更快地激活 Data Cloud 数据
- 连接更多数据源到 Data Cloud
- 提供 Data Cloud 没有的垂直行业模型

### 核心功能

| 功能 | 描述 | 竞争优势 |
|------|------|----------|
| 数据源连接器 | Box, SharePoint, 自定义源接入 Data Cloud | GBase v3.4.5 已有 |
| 数据质量评分 | AI 评估数据完整性、准确性 | 差异化 |
| 智能数据映射 | 自动识别字段对应关系 | 减少实施时间 |
| 实时激活触发 | 基于 Data Cloud 事件触发行动 | 增强价值 |

### ROI 量化
| 指标 | 改善 | 计算方式 |
|------|------|----------|
| Data Cloud 实施时间 | -40% | 自动映射 vs 手工 |
| 数据源覆盖 | +30% | 更多连接器 |
| 数据可用率 | +25% | 质量评分修复 |

### 定价模型
- **按数据量**: ¥X / 百万记录 / 月
- **对标**: 类似 Segment, mParticle 定价

### 优先级评估
| 维度 | 分数 | 理由 |
|------|------|------|
| 市场规模 | 9/10 | $12.96B (2033) |
| 增长率 | 10/10 | 21.7% CAGR |
| 验证程度 | 9/10 | Salesforce 战略重点 |
| 付费意愿 | 9/10 | CDO/CMO 有预算 |
| 竞争强度 | 7/10 | 有竞争但空间大 |
| **总分** | **8.8** | |

## Skill 2: Industry Cloud Accelerator (行业云加速器)

### 市场验证
- 行业云驱动 **60%** 新客户获取
- Financial Services Cloud: 客户留存提升 **20%**
- Manufacturing Cloud: 需求预测提升 **18%**
- 行业专精 ISV 更容易获得融资 (nCino, Veeva 模式)

### 目标行业选择

| 行业 | 市场规模 | GBase 契合度 | 竞争 | 推荐 |
|------|----------|--------------|------|------|
| 金融服务 | 大 | 高 (合规需求) | 高 | ⭐ |
| 医疗健康 | 大 | 中 | 高 | |
| 制造业 | 中 | 高 (供应链) | 中 | ⭐ |
| 零售 | 大 | 高 | 中 | ⭐ |

**推荐优先**: 制造业 + 零售 (供应链场景, GBase 可做差异化)

### 产品定位 (以制造业为例)

**"制造业 Salesforce 的 AI 知识助手"**

| 场景 | 解决问题 | 价值 |
|------|----------|------|
| 供应商知识库 | 采购人员查询供应商信息 | 降低沟通成本 |
| 产品规格查询 | 销售快速获取产品参数 | 加速报价 |
| 合规文档检索 | 快速定位认证、检测报告 | 满足客户审核 |
| 售后知识库 | 服务人员快速解决问题 | 提升首次解决率 |

### 核心功能

| 功能 | 描述 | 行业价值 |
|------|------|----------|
| 行业 RAG 模型 | 预训练的制造业/零售业知识 | 开箱即用 |
| Salesforce 数据联动 | 结合 Account, Product, Case | 上下文相关 |
| 合规检索 | 快速定位认证、审计文档 | 满足行业要求 |
| 多语言支持 | 跨国制造企业需求 | 差异化 (这里多语言才有价值) |

### ROI 量化
| 指标 | 改善 | 行业基准 |
|------|------|----------|
| 首次解决率 | +15% | 服务 KPI |
| 报价时间 | -30% | 销售效率 |
| 合规准备时间 | -50% | 审核成本 |

### 定价模型
- **按用户**: ¥X / 用户 / 月
- **行业套件**: ¥Y / 月 (含行业模型)

### 优先级评估
| 维度 | 分数 | 理由 |
|------|------|------|
| 市场规模 | 8/10 | 60% 新客户来源 |
| 增长率 | 8/10 | 行业云持续增长 |
| 验证程度 | 10/10 | nCino, Veeva 验证 |
| 付费意愿 | 9/10 | 行业买家预算充足 |
| 竞争强度 | 6/10 | 垂直领域竞争较少 |
| **总分** | **8.2** | |

## Skill 3: Data Unification Hub (数据统一中枢)

### 市场验证
- **81%** IT 领导者认为数据孤岛阻碍数字化转型
- **98%** IT 组织报告数字化转型挑战
- 集成超过 5-6 个系统后性能下降
- MuleSoft 被 Salesforce 以 $6.5B 收购

### 买家
| 角色 | 预算 | 购买动机 |
|------|------|----------|
| IT Director | 集成预算 | 减少维护成本 |
| CIO | IT 预算 | 打破数据孤岛 |
| Data Architect | 工具预算 | 简化架构 |

### 产品定位
**"轻量级 Salesforce 数据集成"**

不是替代 MuleSoft，而是：
- 针对中小规模集成 (5-15 系统)
- 更低成本、更快部署
- 专注 Salesforce 生态内的数据统一

### 核心功能

| 功能 | 描述 | vs MuleSoft |
|------|------|-------------|
| 预置连接器 | ERP, HRIS, Marketing 常用系统 | 更快部署 |
| 低代码映射 | 可视化字段映射 | 更易使用 |
| 冲突解决 | AI 识别数据冲突并建议解决 | 差异化 |
| 实时同步 | 双向数据同步 | 基础能力 |

### 典型集成场景

| 源系统 | 目标 | 场景 |
|--------|------|------|
| SAP/Oracle ERP | Salesforce | 订单、库存同步 |
| Workday | Salesforce | 员工数据同步 |
| HubSpot/Marketo | Salesforce | Lead 同步 |
| Shopify/Magento | Salesforce | 电商订单同步 |

### ROI 量化
| 指标 | 改善 | 计算方式 |
|------|------|----------|
| 集成开发时间 | -60% | 预置连接器 |
| 数据一致性 | +90% | 冲突解决 |
| 维护成本 | -40% | 简化架构 |

### 定价模型
- **按连接数**: ¥X / 连接 / 月
- **按数据量**: ¥Y / 百万记录 / 月

### 优先级评估
| 维度 | 分数 | 理由 |
|------|------|------|
| 市场规模 | 8/10 | 81% 企业刚需 |
| 增长率 | 7/10 | 稳定增长 |
| 验证程度 | 10/10 | MuleSoft $6.5B 收购验证 |
| 付费意愿 | 9/10 | IT 有预算 |
| 竞争强度 | 5/10 | MuleSoft 主导但价格高 |
| **总分** | **7.7** | |

## Skill 4: Agentforce Companion (Agentforce 伴侣)

### 市场验证
- Agentic AI 市场: $5-10B → $200B (2034)
- **96%** CIO 计划 2 年内使用 Agentic AI
- AgentExchange: 55 → 122 apps (2025)
- **30%** AI 预算用于 Agentic AI

### 买家
| 角色 | 预算 | 购买动机 |
|------|------|----------|
| CIO | AI 预算 | 扩展 Agent 能力 |
| IT Director | 工具预算 | 填补 Agentforce 空白 |
| Innovation Lead | 创新预算 | 探索新技术 |

### 产品定位
**"让 Agentforce 更强大"**

不是替代 Agentforce，而是：
- 补充 Agentforce 缺失的能力
- 提供 Agentforce 没有的 MCP 工具
- 帮助企业更好地使用 Agentforce

### 补充能力 (基于 Agentforce 真实限制)

| Agentforce 限制 | Companion 补充 | 价值 |
|-----------------|----------------|------|
| 60 秒超时 | 长流程代理执行 | 复杂任务完成 |
| 20 Agent 限制 | Agent 编排层 | 突破限制 |
| MCP 刚起步 (GA Feb 2026) | 成熟 MCP 工具集 | 提前获得能力 |
| 数据质量依赖 | 数据预处理 | 提升 Agent 准确性 |

### 核心功能

| 功能 | 描述 | 市场机会 |
|------|------|----------|
| MCP Tool Library | 预置 Salesforce MCP 工具 | GA 前的时间窗口 |
| Long-running Agent | 突破 60s 限制的后台执行 | 差异化 |
| Agent Analytics | Agent 性能分析和优化 | 管理需求 |
| Knowledge Grounding | RAG 增强 Agent 准确性 | 降低 "自信犯错" |

### ROI 量化
| 指标 | 改善 | 计算方式 |
|------|------|----------|
| Agent 任务完成率 | +40% | 突破超时限制 |
| Agent 准确率 | +25% | RAG 增强 |
| Agent 部署时间 | -50% | 预置工具 |

### 定价模型
- **按 Agent 数**: ¥X / Agent / 月
- **按调用量**: ¥Y / 千次调用

### 风险评估
| 风险 | 概率 | 影响 | 缓解 |
|------|------|------|------|
| Salesforce 原生提供 | 中 | 高 | 持续差异化 |
| 采用不及预期 | 中 | 中 | 多元化产品线 |
| 定价变化 | 高 | 中 | 灵活定价模型 |

### 优先级评估
| 维度 | 分数 | 理由 |
|------|------|------|
| 市场规模 | 10/10 | $200B (2034) |
| 增长率 | 10/10 | ~40% CAGR |
| 验证程度 | 5/10 | 早期市场 |
| 付费意愿 | 6/10 | 待验证 |
| 竞争强度 | 8/10 | 早期少竞争 |
| **总分** | **7.6** | |

## Skill 5: Revenue Intelligence Assistant (收入智能助手)

### 市场验证
- Sales 类应用占 AppExchange **23.44%** (最大类别)
- Clari, Groove, LeanData 等验证市场
- 收入优化工具有明确 ROI

### 买家
| 角色 | 预算 | 购买动机 |
|------|------|----------|
| Sales VP | 销售预算 | 提升业绩 |
| RevOps | 工具预算 | 优化流程 |
| CFO | 可接受预算 | 可量化 ROI |

### 产品定位
**"让销售数据产生洞察"**

| 场景 | 解决问题 | 竞品 |
|------|----------|------|
| 交易预测 | 哪些商机会成交 | Clari |
| 销售问答 | 自然语言查询销售数据 | Einstein |
| 客户洞察 | 客户关系健康度 | Gainsight |
| 知识问答 | 产品/竞品/案例查询 | Guru |

### 差异化定位

选择 **"销售知识问答"** 作为切入点：
- 利用 GBase RAG 能力
- 与 Salesforce 数据结合
- 不与 Clari 等直接竞争

### 核心功能

| 功能 | 描述 | GBase 能力 |
|------|------|------------|
| 产品知识库 | 销售快速查询产品信息 | RAG |
| 竞品情报 | 竞品对比、应对策略 | RAG |
| 案例检索 | 相似客户成功案例 | RAG |
| 销售数据问答 | "本月管道多少" | NL Query |

### ROI 量化
| 指标 | 改善 | 行业基准 |
|------|------|----------|
| 销售准备时间 | -40% | 知识查询效率 |
| 新人上手时间 | -30% | 培训成本 |

## 优先级总结

### 最终排序

| 排名 | Skill | 评分 | 市场验证 | 建议 |
|------|-------|------|----------|------|
| **#1** | Data Cloud Activator | 8.8 | $900M ARR | **立即开始** |
| **#2** | Industry Cloud Accelerator | 8.2 | 60% 新客户 | **立即开始** |
| **#3** | Data Unification Hub | 7.7 | 81% 刚需 | Phase 2 |
| #4 | Agentforce Companion | 7.6 | 高增长早期 | Phase 2 |
| #5 | Revenue Intelligence | 7.1 | 竞争激烈 | Phase 3 |

### vs 之前方案对比

| 之前方案 | 评分 | 新方案 | 评分 | 变化 |
|----------|------|--------|------|------|
| NL Query Assistant | 162 | → Revenue Intelligence (部分) | 7.1 | 降级 |
| Digital Human Agent | 114 | ❌ 移除 (无验证需求) | - | 删除 |
| Data Doctor | 108 | → Data Cloud Activator | 8.8 | 重定位 |
| Voice Salesforce | 96 | ❌ 移除 (加分项非刚需) | - | 删除 |
| AI Coach | 84 | ❌ 移除 (无验证需求) | - | 删除 |

### 关键变化

1. **删除**: 数字人、语音、AI 教练 (无验证市场需求)
2. **重定位**: 数据清洗 → Data Cloud 激活 (对准 $900M 市场)
3. **新增**: 行业垂直、Agentforce 伴侣 (验证的增长点)
4. **保留**: 数据统一、收入智能 (已验证需求)

## Go-to-Market 策略

### 目标客户
1. **首选**: 已使用 Data Cloud 的企业 (已投入, 需要更多价值)
2. **次选**: 正在评估 Agentforce 的企业 (降低风险)
3. **行业**: 制造业、零售业 (供应链场景)

### 定价策略
| 策略 | 理由 |
|------|------|
| 低于 MuleSoft | 中小规模集成市场 |
| 按价值定价 | ROI 可量化 |
| 免费试用 | 降低采用门槛 |

### 渠道
1. **AppExchange** - 主要分发渠道
2. **Salesforce 合作伙伴** - SI 渠道
3. **行业活动** - 垂直行业会议
